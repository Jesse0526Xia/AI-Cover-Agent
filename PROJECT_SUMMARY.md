# AI音频Agent 项目总结

## 项目概述

AI音频Agent是一个智能的音频处理助手，基于LangChain框架构建，能够理解自然语言指令并执行各种音频处理任务。

## 核心功能

### 1. 音色管理 🎤
- **音色克隆**: 从音频中提取音色特征并保存到仓库
- **音色仓库**: 管理多个音色，支持搜索、预览和编辑
- **音色转换**: 用指定音色替换音频中的原始音色

### 2. 音调控制 🎵
- **音调调节**: 支持升调/降调（±12半音）
- **快速预设**: 性别转换、八度调节、小黄人等
- **音高检测**: 分析音频的音高信息

### 3. 语音合成 🗣️
- **文字转语音**: 用指定音色朗读文本
- **情感控制**: 支持8种情感（开心、悲伤、愤怒等）
- **批量合成**: 一次处理多个文本

### 4. AI歌唱 🎶
- **AI作曲**: 自动生成歌曲旋律
- **歌声合成**: 用指定音色演唱歌曲
- **歌词生成**: 根据主题生成歌词
- **多风格支持**: 流行、摇滚、民谣等

### 5. 情感控制 🎭
- **情感转换**: 调整音频的情感表达
- **强度控制**: 低、中、高三个强度级别
- **情感检测**: 识别音频中的情感

## 技术架构

### Agent框架
- **LangChain**: Agent编排和工具调用
- **OpenAI GPT**: 自然语言理解和生成
- **记忆管理**: 对话历史和用户偏好

### 音频处理
- **librosa**: 音频分析和处理
- **soundfile**: 音频读写
- **pydub**: 音频编辑

### 音频模型
- **RVC**: 音色转换（待集成）
- **Edge-TTS**: 语音合成（待集成）
- **AudioLDM**: 音乐生成（待集成）

### Web界面
- **Flask**: Web服务器
- **HTML/CSS/JS**: 前端界面
- **REST API**: 后端接口

## 使用方法

### 快速启动

1. **安装依赖**
```bash
pip install -r requirements.txt
```

2. **配置API密钥**
编辑 `config/config.yaml`，设置OpenAI API密钥

3. **运行程序**
```bash
# 命令行模式
python src/main.py

# Web界面模式
python src/web/app.py
```

### 使用示例

**音色克隆:**
```
用户: 帮我克隆这个音频里的音色
Agent: 好的，我来提取这个音频的音色特征...
```

**音色转换:**
```
用户: 用voice_001的音色转换这段音频
Agent: 正在执行音色转换...
```

**音调调节:**
```
用户: 把这段音频升高3个半音
Agent: 音调调节完成...
```

**语音合成:**
```
用户: 用开心的情感朗读"你好世界"
Agent: 正在生成语音...
```

**AI歌唱:**
```
用户: 用流行风格唱"春天来了"
Agent: 正在生成歌曲...
```
